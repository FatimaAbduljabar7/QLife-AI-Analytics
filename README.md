# QLife AI Analytics
**Overview**

AI-powered healthcare analytics project using machine learning and data analysis techniques.

### **Features**

- Data preprocessing
- Machine learning model comparison
- analysis
- Visualization of actual vs predicted values

### **Models Used**
- Random Forest
- XGBoost
- Linear Regression
- Ridge Regression
- KNN
- Decision Tree
